## 📌 Description

The **"Videos Chapters & Logos"** plugin allows you to embed MP4 videos with a logo, poster, and interactive markers on your WordPress site.

With this plugin, you can:

- Embed MP4 videos in posts or pages  
- Display a custom logo over the video  
- Add an optional poster image  
- Define chapter markers inside the video  
- Allow users to click markers to jump to specific timestamps
